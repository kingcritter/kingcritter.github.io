TextFormatter

Have you ever had some text you wanted to squeeze into a particular shape? Well, now you can.

The code lives in its github repository; read on for more info and examples.

What this thing does

You give this program two things: a png with a transparent background, and a text file. The program arranges your text in the shape of the image given. You can specify the resolution of the output; that is, how many pixels is equated to a character.

Why this thing exists

Was doing a tshirt design for my university's ACM chapter, and wanted to have a bunch of code in the shape of our school's mascot. I couldn't find a program to do this, so I made one.

Usage and Installation

TextFormatter uses the image processing library Pillow. In Fedora, it's in the repositories as python-pillow; I imagine other distros are similar.

Usage example (note that it's a python3 application):

python3 -i /path/to/image.png -t /path/to/textfile.txt

Use the -h flag to get help:

$ python3 TextFormatter.py -h Usage: TextFormatter.py [options] Options: -h, --help show this help message and exit -i IMAGE, --image=IMAGE specifiy input image -t TEXTFILE, --text=TEXTFILE specify text file -r X Y, --resolution=X Y specify resolution of characters, x and y --html output is html with color codes -f OUTPUTFILE, --file=OUTPUTFILE specify output file

Only -i and -t are required. Output is printed to the console by default, and the default resolution is 10 by 20.
